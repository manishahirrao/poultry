# Logistics Checklist
**Setup, teardown, and follow-up process for mandi events**

## Pre-Event Planning (1 Week Before)
### Venue Coordination
- [ ] Confirm mandi location and contact person
- [ ] Get permission for banner placement
- [ ] Identify backup indoor location (feed shop, community center)
- [ ] Check parking availability for team
- [ ] Verify electricity access (if needed)

### Team Coordination
- [ ] Assign roles (Technical lead, Relationship builder)
- [ ] Confirm team availability for event date
- [ ] Share event details with team (location, timing, script)
- [ ] Conduct brief training session on objections handling
- [ ] Exchange emergency contact numbers

### Materials Preparation
- [ ] Order banner printing (allow 3-5 days lead time)
- [ ] Print event scripts and checklists (2 copies each)
- [ ] Prepare QR code stand/tablet holder
- [ ] Pack marketing collateral (one-pagers, business cards)
- [ ] Charge all devices (phones, tablets, power banks)
- [ ] Test WhatsApp demo flow on all devices
- [ ] Load latest prediction data into app
- [ ] Prepare signup forms (digital and paper backup)

### Logistics
- [ ] Arrange transportation (car/bike for team)
- [ ] Plan route to mandi (check traffic patterns)
- [ ] Pack refreshments (water, snacks for team)
- [ ] Prepare first aid kit
- [ ] Check weather forecast (have rain plan ready)
- [ ] Inform local contacts about arrival time

## Day-Before Preparation
### Final Checks
- [ ] Recharge all devices to 100%
- [ ] Download offline maps for mandi location
- [ ] Print backup materials (scripts, checklists)
- [ ] Pack emergency kit (charger cables, power bank)
- [ ] Confirm team arrival time (7:30 AM at mandi)
- [ ] Set alarm for early departure
- [ ] Check WhatsApp Business API status
- [ ] Verify prediction data is current

## Event Day Timeline

### 6:00 AM - Team Departure
- [ ] Team leaves from base location
- [ ] All materials loaded in vehicle
- [ ] Final check of equipment list

### 7:00 AM - Arrival at Mandi
- [ ] Arrive at mandi location
- [ ] Contact local mandi official
- [ ] Identify optimal banner placement spot
- [ ] Scout for quiet interview location

### 7:30 AM - Setup
- [ ] Hang banner at eye level (5ft from ground)
- [ ] Set up table with demo devices
- [ ] Position QR code stand
- [ ] Test QR code scannability
- [ ] Start WhatsApp demo on tablet (loop mode)
- [ ] Position team members for coverage
- [ ] Take setup photos for documentation

### 8:00 AM - 12:00 PM - Event Execution
- [ ] Engage farmers with opening script
- [ ] Conduct prediction verification (hourly)
- [ ] Capture video testimonials
- [ ] Handle objections using script
- [ ] Collect signup information
- [ ] Monitor device battery levels
- [ ] Take progress photos every hour
- [ ] Track engagement metrics

### 12:00 PM - Teardown
- [ ] Thank mandi officials
- [ ] Carefully remove banner
- [ ] Pack all materials
- [ ] Collect trash (leave location clean)
- [ ] Take final team photo
- [ ] Depart from mandi

## Risk Management
### Weather Contingency
- **Light rain**: Continue under umbrella/tarp
- **Heavy rain**: Move to backup indoor location
- **Extreme heat**: Shift to morning slot (7-11 AM)
- **Cold weather**: Provide warm beverages for team

### Low Attendance Contingency
- Move to higher-traffic area within mandi
- Offer immediate demo on farmer's phone
- Partner with local feed shop for co-location
- Reschedule to different day/time

### Technical Failure Contingency
- Use offline demo (cached prediction data)
- Switch to manual signup (paper forms)
- Use personal hotspot if no internet
- Have backup devices ready

### Safety Considerations
- [ ] Team members carry emergency contacts
- [ ] Avoid isolated areas
- [ ] Keep valuables secure
- [ ] Have vehicle nearby for quick exit
- [ ] First aid kit accessible
- [ ] Emergency cash available
