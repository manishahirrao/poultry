# Prediction Verification Checklist
**Daily verification process for mandi events**

## Pre-Event Setup (Day Before)
- [ ] Confirm mandi location and timing
- [ ] Check weather forecast (have backup plan for rain)
- [ ] Charge all devices (phones, tablets, power bank)
- [ ] Load latest prediction data into app
- [ ] Print event checklist and script
- [ ] Pack banner, QR code stand, marketing materials
- [ ] Test WhatsApp demo flow end-to-end
- [ ] Verify internet connectivity at mandi location
- [ ] Inform local contact about arrival time

## On-Site Setup (Morning Of)
- [ ] Arrive at mandi by 7:30 AM (30 min before event)
- [ ] Identify high-traffic location near entrance
- [ ] Set up banner at eye level (5ft from ground)
- [ ] Test QR code scannability with multiple phones
- [ ] Set up table with tablets/demo devices
- [ ] Position team members for coverage
- [ ] Have water and snacks for team
- [ ] Start WhatsApp demo on tablet (loop mode)

## Accuracy Calculation Formula
```
Directional Accuracy = (Correct Direction Predictions / Total Predictions) × 100

Where "Correct Direction" means:
- If prediction says price will go UP → actual price went UP or stayed same
- If prediction says price will go DOWN → actual price went DOWN or stayed same

Price Accuracy = 1 - (|Predicted P50 - Actual Price| / Actual Price) × 100
```

## Troubleshooting
### No internet connectivity
- [ ] Use cached prediction data from app
- [ ] Switch to mobile hotspot
- [ ] Continue with offline demo
- [ ] Collect phone numbers for manual follow-up

### Banner setup issues
- [ ] Use alternative roll-up banner
- [ ] Position QR code on tablet instead
- [ ] Have team member manually share link

### Low farmer engagement
- [ ] Move to higher-traffic area
- [ ] Offer immediate demo on their phone
- [ ] Share testimonials more aggressively
- [ ] Use local language/dialect more

### Weather issues
- [ ] Move to covered area (feed shop nearby)
- [ ] Use indoor location at partner location
- [ ] Reschedule if severe weather

## Success Criteria
- [ ] 50+ farmer engagements
- [ ] 20+ demo signups
- [ ] 5+ video testimonials
- [ ] Prediction accuracy >90%
