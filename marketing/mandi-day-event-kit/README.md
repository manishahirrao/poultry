# Mandi Day Event Kit
**Marketing Initiative #2: Live Prediction Verification at Local Mandis**

## Overview
Live prediction verification events at major mandis on Monday mornings when farmers sell broilers. This builds trust through public validation of PoultryPulse AI's 7-day price predictions.

## Event Details
- **Timing**: Monday mornings, 8:00 AM - 12:00 PM (peak selling hours)
- **Locations**: Gorakhpur, Deoria, Kushinagar, Basti, Maharajganj mandis
- **Team**: 2 people (1 technical, 1 relationship builder)
- **Duration**: 4-8 weeks (rotating through 5 mandis)

## Materials Included
1. **Banner Design Specs** - Physical banner for mandi setup
2. **Event Script** - Hindi/English talking points for engaging farmers
3. **Prediction Verification Checklist** - Daily verification process
4. **Video Capture Guide** - How to film farmer testimonials
5. **Logistics Checklist** - Setup, teardown, and follow-up process

## Expected Outcomes
- 50-100 farmer signups per event
- Powerful social proof content for WhatsApp sharing
- Real-time accuracy validation
- Word-of-mouth spread through mandi networks

## Success Metrics
- Number of farmer signups
- Video testimonials captured
- Prediction accuracy verified on-site
- WhatsApp shares of event content
