# Banner Design Specs
**Physical banner for mandi setup**

## Dimensions
- **Size**: 6ft x 3ft (180cm x 90cm)
- **Material**: PVC vinyl (weather-resistant)
- **Finish**: Matte (avoid glare in sunlight)
- **Grommets**: 4 corners with metal grommets for easy hanging

## Design Layout

### Top Section (30% height)
**Background**: PoultryPulse brand color (green #10B981)

**Text (Hindi - Large, Bold)**:
```
🐔 पहले से पता चलेगा — बेचने से पहले
Know Before You Sell
```

### Middle Section (40% height)
**Background**: White

**Text (Hindi - Medium)**:
```
7 दिन का भाव अनुमान • 95%+ सटीकता
7-Day Price Forecast • 95%+ Accuracy

आज का भाव: ₹164/kg (₹158–₹170)
Today's Price: ₹164/kg (₹158–₹170)
```

**Visual**: Simple line chart showing 7-day prediction (upward trend arrow)

### Bottom Section (30% height)
**Background**: PoultryPulse brand color (green #10B981)

**Text (Hindi - Medium)**:
```
📱 WhatsApp Demo आज़माएं
Scan QR Code → Free 14-Day Trial

📞 +91 XXXXX XXXXX
🌐 poultrypulse.ai
```

**QR Code**: Large, centered, links to /try-whatsapp

## Color Palette
- **Primary Green**: #10B981 (Tailwind emerald-500)
- **Secondary Dark**: #064E3B (Tailwind emerald-900)
- **Text White**: #FFFFFF
- **Text Black**: #000000
- **Accent Orange**: #F59E0B (for price highlights)

## Typography
- **Hindi Headline**: Noto Sans Devanagari, Bold, 48pt
- **English Subtitle**: Inter, Semibold, 32pt
- **Body Text**: Noto Sans Devanagari + Inter, Regular, 24pt
- **Numbers**: Inter, Bold, 36pt

## Visual Elements
- **Logo**: PoultryPulse AI logo (top right corner, 4" x 4")
- **Chart Icon**: Simple line graph with confidence bands
- **WhatsApp Icon**: Green WhatsApp logo (3" x 3")
- **Phone Icon**: Simple handset icon (2" x 2")

## Printing Specifications
- **Resolution**: 150 DPI minimum
- **Color Mode**: CMYK
- **Bleed**: 0.5" on all sides
- **File Format**: PDF with fonts outlined
- **Print Vendor**: Local print shop with large format capability

## Setup Instructions
1. Hang banner at eye level (5ft from ground)
2. Position near mandi entrance or high-traffic area
3. Ensure QR code is scannable (test with phone)
4. Use bungee cords or rope for secure hanging
5. Have team member stationed near banner for questions

## Alternative: Roll-up Banner
If space is limited, use roll-up banner (33" x 81"):
- Same design, scaled down
- Easier transport and setup
- Can be used indoors at feed shops
