## CMS Checklist (per piece, before publishing)

- [ ] Frontmatter metadata complete (title, slug, author, tags, keywords)
- [ ] Hero stat/pull-quote added for article header
- [ ] Featured image assigned (FlockIQ brand style — forest green + data visual)
- [ ] Internal links added (minimum 2 per piece)
- [ ] Schema markup applied (Review schema for case studies, Article schema for blogs)
- [ ] Hindi content reviewed for Devanagari rendering (UTF-8)
- [ ] OpenGraph image generated (/api/og with title + hero stat)
- [ ] Indexed in sitemap.xml
- [ ] Social share scheduled (LinkedIn for global, WhatsApp for India)
