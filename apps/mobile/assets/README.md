# Asset Files

This directory contains static assets for the mobile app.

## Required Assets

### Icons
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Adaptive icon (1024x1024)
- `favicon.png` - Web favicon (192x192)

### Splash Screen
- `splash.png` - Splash screen image (1284x2778)

### Fonts
See `fonts/README.md` for font requirements.

## Asset Sources

Assets should follow the PoultryPulse AI brand guidelines:
- Primary color: #1A6B3C (brandGreen700)
- Secondary color: #E8621A (saffronOrange)
- Typography: Noto Sans Devanagari for Hindi, Sora for English numbers
