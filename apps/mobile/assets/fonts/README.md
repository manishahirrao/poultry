# Font Assets

This directory should contain the Noto Sans Devanagari font files for Hindi text rendering.

## Required Fonts

- `NotoSansDevanagari-Regular.ttf` - Weight 400
- `NotoSansDevanagari-Medium.ttf` - Weight 500
- `NotoSansDevanagari-SemiBold.ttf` - Weight 600
- `NotoSansDevanagari-Bold.ttf` - Weight 700

## Download

Download from Google Fonts:
https://fonts.google.com/specimen/Noto+Sans+Devanagari

## Installation

Place the font files in this directory with the exact names specified above.
