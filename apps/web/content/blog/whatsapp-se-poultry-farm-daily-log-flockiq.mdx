<h2>संक्षेप में:</h2>
<p>FlockIQ हर रोज़ शाम को आपके WhatsApp पर एक message भेजता है। आप उसका reply करते हैं — आज कितने पक्षी मरे, कितना feed दिया। बस। FlockIQ automatically आपके farm का daily log भर देता है। Integrator को call करने की ज़रूरत नहीं। App download नहीं। Form नहीं। सिर्फ WhatsApp।</p>

<hr>

<h2>यह feature क्यों बना?</h2>

<p>हम भारत के poultry farmers से बात करते हैं तो एक बात बार-बार आती है:</p>

<blockquote>
<p><em>"Data भरना बहुत झंझट है। दिन भर काम करके शाम को कोई app खोलकर form भरने का मन नहीं होता।"</em></p>
</blockquote>

<p>और integrators की तरफ से:</p>

<blockquote>
<p><em>"मेरे 8 farms हैं। हर रोज़ 8 किसानों को phone करो, data लो, excel में डालो — इसमें 3-4 घंटे चले जाते हैं।"</em></p>
</blockquote>

<p>यही दोनों problems एक साथ हल करता है <strong>WhatsApp Daily Log Automation।</strong></p>

<h2>यह काम कैसे करता है? Step-by-Step</h2>

<h3>Step 1: Setup (एक बार, 10 मिनट)</h3>

<p><strong>अगर आप किसान हैं:</strong></p>
<ul>
<li>आपका integrator आपका WhatsApp number FlockIQ में add करता है</li>
<li>Reminder time set होती है (default: शाम 6:00 बजे)</li>
<li>Language: Hindi या English</li>
<li>एक test message आता है → आप confirm करते हैं → हो गया।</li>
</ul>

<p><strong>अगर आप integrator हैं:</strong></p>
<ul>
<li>FlockIQ dashboard पर जाएँ → WhatsApp Setup</li>
<li>प्रत्येक farm के लिए: farmer का number + farm name + batch details + reminder time</li>
<li>Farmer को test message → confirmation → Active</li>
</ul>

<h3>Step 2: रोज़ शाम को message आता है</h3>

<p>हर रोज़ शाम को — आपके set किए हुए time पर — FlockIQ का message आता है:</p>

<pre><code>नमस्ते राजेश जी! 🐔 FlockIQ Daily Log
Farm: जंगल कौड़िया | Batch #14 | Day 22

आज का data भेजें:

Example: "3 dead | 1250 kg feed | weight 920g"

कोई problem है? जैसे: "3 dead | 1250 feed | 920 weight | सांस की problem"

आपका reply automatically dashboard पर जाएगा।
</code></pre>

<h2>किसान को क्या फायदा?</h2>

<h3>कोई app download नहीं</h3>

<p>आपके पास पहले से WhatsApp है। कोई नया app नहीं, कोई login नहीं, कोई password नहीं।</p>

<h3>Integrator की रोज़ की call नहीं</h3>

<p>पहले integrator रोज़ call करता था — "कितने मरे? कितना दाना दिया?" अब वह call बंद हो गई। आप जब तैयार हों, शाम को reply करें।</p>

<h3>आपका record automatically बनता है</h3>

<p>Batch close होने पर पूरे batch का record automatically तैयार है — हर दिन का mortality, feed, FCR। अगर कभी dispute हो या performance review हो, आपके पास complete data है।</p>

<h3>Voice note भी support (coming soon)</h3>

<p>अभी text reply करना होता है। जल्द ही voice note support आएगा — जिसमें आप बोल सकेंगे और FlockIQ automatically transcript करेगा।</p>

<h3>Data Completeness: 61% → 96%</h3>

<p>Phone calling system में data अक्सर incomplete रहता था:</p>
<ul>
<li>कोई farmer call नहीं उठाता</li>
<li>कोई numbers याद नहीं रखता</li>
<li>कोई feed weight measure नहीं करता ("लगभग 1200 किलो था")</li>
</ul>

<p>WhatsApp automation में:</p>
<ul>
<li>Farmer अपने time पर reply करता है — सुबह, दोपहर, शाम — जब convenient हो</li>
<li>System 2–3 घंटे बाद gentle reminder भेजता है अगर reply नहीं आया</li>
<li>Data completeness significantly improve होती है</li>
</ul>

<h3>Early Problem Detection</h3>

<p>जब रोज़ का data accurate आता है, तो problems 5–7 दिन पहले दिखती हैं।</p>

<p><strong>Example:</strong><br>
Farm पर Day 14–16 में mortality 0.2% से 0.5% प्रति दिन हो गई — यह dashboard पर तुरंत दिखा। Integrator ने Day 17 को farm visit किया। Subclinical respiratory infection मिली। Treatment Day 18 से शुरू। Batch ने recover किया।</p>

<p>बिना daily log के: Day 22–24 तक पता चलता, tab तक बहुत देर हो जाती।</p>

<h2>System कितना Smart है?</h2>

<h3>Natural Language Understanding</h3>

<p>FlockIQ का parser 6,000+ real farmer messages पर trained है। यह handle करता है:</p>
<ul>
<li>Hindi in Devanagari: "आज 3 मरे, 1250 किलो दाना"</li>
<li>Hindi in Roman: "aaj 3 mare, 1250 kg dana"</li>
<li>Mixed: "3 dead aaj, 1250 dena"</li>
<li>Abbreviations: "3d 1250f" (3 dead, 1250 feed)</li>
<li>Numbers only: "3 1250" (when context is clear)</li>
<li>Urdu/Arabic numerals: automatically converted</li>
</ul>

<h3>Ambiguity Handling</h3>

<p>अगर reply unclear हो, system politely पूछता है:</p>

<pre><code>Rajesh ji, ek baat confirm karein:
Feed kitna tha? Sirf number bhejein (jaise: 1250)
</code></pre>

<h3>Smart Skip Logic</h3>

<p>Weekend पर या batch placement/close के दिन system automatically adjust करता है।</p>

<p>अगर farmer ने batch pause किया (vaccinations, drug withdrawal period) — integrator mark कर सकता है, उन दिनों reminder skip होती है।</p>

<h2>Setup: Integrators के लिए Step-by-Step</h2>

<p><strong>Per farm:</strong></p>
<ol>
<li>Farmer का WhatsApp number enter करें</li>

<p>अगर 5+ farms हैं → CSV upload करें (number, name, time, language) → सब एक साथ setup हो जाता है।</p>
