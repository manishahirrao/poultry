# Static Assets for PoultryPulse AI

This directory should contain the following static assets for SEO and PWA support:

## Required Assets

1. **favicon.ico** (16x16, 32x32)
   - Favicon for browser tabs
   - Place in this directory as `favicon.ico`

2. **apple-touch-icon.png** (180x180)
   - iOS home screen icon
   - Place in this directory as `apple-touch-icon.png`

3. **icon-192.png** (192x192)
   - Android home screen icon
   - Referenced in manifest.json
   - Place in this directory as `icon-192.png`

4. **icon-512.png** (512x512)
   - Android home screen icon (high res)
   - Referenced in manifest.json
   - Place in this directory as `icon-512.png`

5. **og-image.png** (1200x630)
   - Open Graph image for social sharing
   - Recommended format: PNG with 1.91:1 aspect ratio
   - Place in this directory as `og-image.png`

6. **logo.png** (512x512 or larger)
   - Company logo for schema markup
   - Place in this directory as `logo.png`

## Design Guidelines

- Use brand color: #166534 (brandGreen-700)
- Include poultry/chicken imagery or abstract AI patterns
- Ensure high contrast for readability
- Use transparent backgrounds where appropriate
- Optimize file sizes (favicon < 50KB, icons < 200KB, OG image < 500KB)

## Tools for Generation

- Favicon: https://realfavicongenerator.net/
- PWA Icons: https://www.pwabuilder.com/imageGenerator
- OG Image: Canva or similar design tool
